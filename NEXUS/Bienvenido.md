# Bienvenido a la bóveda NEXUS

Esta carpeta es la base documental/memoria para notas de conocimiento durable.

## Para qué se usa

- guardar memorias de arquitectura, decisiones y lecciones reutilizables;
- estandarizar cierres de sesión y notas de aprendizaje;
- mantener trazabilidad pedagógica del flujo `Change / Reason / Concepts / Practice`.

## Estructura recomendada

```text
NEXUS/
  _index.md
  _templates/
    memory-entry.md
    session-close.md
  <project>/
    memories/
    axioms/
    sessions/
```

## Regla práctica

Guardar aquí “verdades de mañana” (decisiones durables), no ruido temporal (logs crudos o texto efímero).
